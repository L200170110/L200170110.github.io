Sistem Administrasi Basis Data

Tabel Perintah DDL (Data Definition Language)

No Perintah Fungsi
1 Create Database Membuat database baru
2 Create Table Membuat tabel baru pada database
3 Alter Table Merubah struktur dari sebuah tabel
4 Drop Database Menghapus database
5 Drop Table Menghapus tabel pada database
6 Rename merubah dan mengedit nama pada database

Tabel Perintah DML (Data Management Language)

No Perintah Fungsi
1 Update Memperbaharui isi data dari sebuah tabel dan database
2 Insert Menyisipkan dan memasukkan data baru ke dalam sebuah database dan tabel
3 Select Mengambil dan juga menampilkan data yang terdapat di dalam tabel atau database
4 Delete Menghapus data yang terdapat di dalam tabel ataupun database
